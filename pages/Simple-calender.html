Project info

  • Title: Simple-Calendar

  • Project Description: his is a basic calendar application where the current timeslots refer to a normal 9-5 workday. The current time is highlighted in red while the past is in grey, anything in the future is all highlighted in green. Each of the slots is accompanied by a save button that stores the content into a localstorage function and then displays that saved content when the page is refreshed.
